## :computer: How to run

Clone the repository:

```
$ git clone https://github.com/buivantan01/TypeScript.git
```

Enter the directory:

```
$ cd TypeScript
```

Install the dependencies:

```
$ npm install -g typescript
```

Next

```
$ npm install --save-dev nodemon
```

Next

```
$ tsc -w index
```


Next

```
$ npm run watch
```
